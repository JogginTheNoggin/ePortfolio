Artifact 1: Android UI/UX Improvements

A Modern, Simplisitic and Responsive UI Design

Narrative

The artifact involved in the milestone is an android application designed to allow users to track their progress towards a weight goal. It was initially developed during the summer of 2024 for a mobile development course. The updates made are based on UI improvements, this not only shows the ability to make iterative improvements but also showcase the ability to adhere to styling conventions. This can be used via the changes made to the home page, a login page, which have adopted a consistent, modern, yet simplistic design. Cohesion between all UI elements is important for professionalism and ensuring the theme is felt by the users.

As a result the following outcome has been achieved:

  1. Demonstrate an ability to use well-founded and innovative techniques, skills, and tools in computing practices to implement computer solutions that deliver value and accomplish industry-specific goals.

In regards to the process of making the needed improvement, the first process was looking for design inspirations. These designs were assessed based on creativity, consistency and feasibility given the time constraints. By narrowing them down, a modern, simplistic look was chosen. Custom Background elements were the main contributor in fulfilling the design vision. Getting these designs to integrate into previous work was the biggest challenge, as redesigning the layout had, in multiple instances, alternated the functionality of the application, in turn breaking it.
